# Material Calc
A PWA (Progressive Web App) calculator inspired by google's material you design, tailored to offer a more fitting experience into chrome os, 

## Why does it even exist and why a random calculator
Well, to be honest this project was born mostly cause i saw chrome os's calculator and thought. why google doesnt add material design to the calculator app? it just looks so dated (and since it's just a pwa to a functioning domain, I wanted to replicate it) but since i currently don't have the knowledge to code on my own. I decided to code this purely with AI, and also learn how to navigate and use Github and Github Pages.

## Key points

1.- This is purely a **calculator**, there's not much to add to it appart from the design and animations.

2.- It can actually run **offline**, just click "install" on your dropdown menu (tested on Chrome and ChromeOS) it should run without any wifi connection.

3.- I will actively improve it adding design tweaks and maybe some light features (mostly aesthetic improvements).
